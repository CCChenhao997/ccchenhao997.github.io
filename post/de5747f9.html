Gated-Attention Readers for Text Comprehension

ACL 2017 [paper]

Bhuwan Dhingra ∗, Hanxiao Liu ∗, Zhilin Yang, William W. Cohen, Ruslan Salakhutdinov

School of Computer Science, Carnegie Mellon University

Introduction

Our model, the Gated-Attention (GA) Reader , integrates a multi-hop architecture with a novel attention mechanism, which is based on multiplicative interactions between the query embedding and the intermediate states of a recurrent neural network document reader. This enables the reader to build query-speciﬁc representations of tokens in the document for accurate answer selection.

The success of many recent models can be attributed primarily to two factors:

  1. Multi-hop architectures, allow a model to scan the document and the question iteratively for multiple passes.
  2. Attention mechanisms, allow the model to focus on appropriate subparts of the context document.

We propose a ﬁner-grained model which attends to components of the semantic representation being built up by the GRU. The new attention mechanism, called gated-attention, is implemented via multiplicative interactions between the query and the contextual embeddings, and is applied per hop to act as ﬁne-grained information ﬁlters during the multi-step reasoning. The ﬁlters weigh individual components of the vector representation of each token in the document separately.
