Attention Modeling for Targeted Sentiment

ACL 2017 [paper]

Jiangming Liu and Yue Zhang

Singapore University of Technology and Design

Introduction

先前的工作着重于学习用于给定输入句子的目标特定的表示 (representation) ，以用于分类。然而，它们不能够对于每个词在句子中关于目标情感极性的贡献 (contribution) 进行显式地建模。

为此，我们研究了一个注意力模型。特别地，一个普通的LSTM被用来归纳整个句子的注意值。在前人工作的基础上，进一步扩展了模型以区分给定目标的左右上下文。

Model

我们使用一个 bidirectional LSTM 去表示输入的词序列 w0,w1,...,wnw_0,w_1,...,w_nw0​,w1​,...,wn​ 作为隐藏结点 h0,h1,...,hnh_0,h_1,...,h_nh0​,h1​,...,hn​：

[h0;...;hn]=BILSTM([w0;...;wn])[h_0;...;h_n]=BILSTM([w_0;...;w_n]) [h0​;...;hn​]=BILSTM([w0​;...;wn​])

目标词表示为 hth_tht​ ，是将目标短语 [ht0;...;htm][h_{t_0};...;h_{t_m}][ht0​​;...;htm​​] 做平均得到的结果。

我们提出了三种注意力的变体去建模上下文词与目标词的关系。

Vanilla Model

我们使用一个普通的注意力模型计算关于句子中每个词的一个权重值 α\alphaα ，句子 sss 的最终表示为：

s=attention([h0;...;hn],ht)=∑inαihiαi=exp(βi)∑jnexp(βj)s = attention([h_0;...;h_n],h_t)=\sum_i^n \alpha_i h_i \\\alpha_i = \frac{exp(\beta_i)}{\sum_j^n exp(\beta_j)} s=attention([h0​;...;hn​],ht​)=i∑n​αi​hi​αi​=∑jn​exp(βj​)exp(βi​)​

权重分数 β\betaβ 通过目标表示和上下文词表示计算得到：

βi=UTtanh(W1⋅[hi;ht]+b1)\beta_i = U^T tanh(W_1 · [h_i;h_t]+b_1) βi​=UTtanh(W1​⋅[hi​;ht​]+b1​)

然后，句子表示 sss 被用于预测情感标签的概率分布 ppp :

p=softmax(W2s+b2)p = softmax(W_2 s + b_2) p=softmax(W2​s+b2​)

我们将这种典型的模型称为BILSTM-ATT。

Contextualized Attention

我们对普通的注意力方法做了两个扩展。第一个是上下文注意力模型 (BILSTM-ATT-C) ，其中句子根据目标词被分成两个部分，即左上下文和右上下文，注意力被分别作用于左右上下文。

具体地，左上下文的表示：

sl=attention([h0;...;ht0−1],ht)s_l = attention([h_0;...;h_{t_0-1}], h_t) sl​=attention([h0​;...;ht0​−1​],ht​)

右上下文的表示：

sr=attention([htm+1;...;hn],ht)s_r = attention([h_{t_m+1};...;h_n], h_t) sr​=attention([htm​+1​;...;hn​],ht​)

然后和之前的表示 sss 一起预测情感标签的分布：

p=softmax(W1s+Wlsl+Wrsr+b1)p=softmax(W_1s + W_l s_l + W_r s_r + b_1) p=softmax(W1​s+Wl​sl​+Wr​sr​+b1​)

Contextualized Attention with Gates

第二个扩展是增加 gate 来控制上下文信息流 (BILSTM-ATT-G) 。这是因为情感信号可以被左上下文、右上下文或整个句子所主动。

使用了三个由目标词和对应的上下文所控制的 gates：

z∝exp(W1s+U1ht+b1)zl∝exp(W2sl+U2ht+b2)zr∝exp(W3sr+U3ht+b3)z \varpropto exp(W_1 s + U_1 h_t + b_1) \\z_l \varpropto exp(W_2 s_l + U_2 h_t + b_2) \\z_r \varpropto exp(W_3 s_r + U_3 h_t + b_3) z∝exp(W1​s+U1​ht​+b1​)zl​∝exp(W2​sl​+U2​ht​+b2​)zr​∝exp(W3​sr​+U3​ht​+b3​)

z+zl+zr=1→z + z_l + z_r = \overrightarrow{1}z+zl​+zr​=1 ，s,sl,srs, s_l, s_rs,sl​,sr​ 之间的线性融合表示为：

s~=z⊙s+zl⊙sl+zr⊙sr\widetilde{s} = z \odot s + z_l \odot s_l + z_r \odot s_r s=z⊙s+zl​⊙sl​+zr​⊙sr​

情感标签的预测分布为：

p=softmax(W4s~+b4)p=softmax(W_4 \widetilde{s} + b_4) p=softmax(W4​s+b4​)
