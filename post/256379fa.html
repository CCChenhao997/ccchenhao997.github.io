Hierarchical Attention Based Position-aware Network for Aspect-level Sentiment Analysis

CoNLL 2018 [paper] [code]

Lishuang Li, Yang Liu and AnQiao Zhou

School of Computer Science and Technology, Dalian University of Technology

Introduction

之前的工作已经证明了 aspects 与 contexts 之间的交互是很重要的。基于此，本文提出了一种简洁的基于层次注意的机制，以融合 targets 和上下文词的信息。此外，大多数的方法在编码句子的时候忽略了 aspect 的位置信息，在本文中，我们认为位置感知的表示对于该任务是有益的。因此，我们提出了一种 hierarchical attention based position-aware network (HAPN)

Model

我们假设一个句子由 nnn 个词 w={w1,w2,...,wn}w=\{w_1,w_2,...,w_n\}w={w1​,w2​,...,wn​} 和 一个 target 具有 mmm 个词 wt={w1t,w2t,...,wmt}w^t=\{w_1^t,w_2^t,...,w_m^t\}wt={w1t​,w2t​,...,wmt​} 。wtw^twt 是 www 的一个子序列。

我们的模型主要包含四个部分：input embeddings，Bi-GRU based encoding layer，hierarchical attention based fusion layer，the output layer。

Input Embedding

Embedding 层主要有两个部分：the word embeddings 和 the position embeddings。

与 word embedding 层类似，the position embedding 层是一个 Wp∈Rdp×vpW_p \in \mathbb{R}^{d_p \times v_p}Wp​∈Rdp​×vp​ ，dpd_pdp​ 是 position embeddings 的维度，vpv_pvp​ 是每个词与target之间相对位置的数量。The position embedding lookup table 是随机初始化的，在训练阶段调整。

在 encoding 阶段，我们首先将每个 token wiw_iwi​ 转换为一个实值向量 xix_ixi​ ，具体做法就是将 word embedding 与 position embedding 拼接起来。

Hierarchical Attention Based Fusion Layer

首先，通过 source2aspect attention 捕获 target words 中最重要的线索，获得 aspect representation。

随后，基于 aspect representation 和 encoded position-aware representation 生成了每个词的 aspect-specific representation。

然后，使用 source2context attention 捕获上下文中最具指示意义的情感词，并生成加权求和后的 embeddings 作为句子的最终表示。

Source2aspect Attention

由于大多数的 aspects 包含至少两个及以上的词，我们引入一个 source2aspect 机制来生成 aspect 的表示。首先，我们引入一个 score function，将 target 中每个词的 word embeddings 作为输入：

ft(wit)=tanh(Wt⋅wit)f_t(w_i^t)=tanh(W_t · w_i^t) ft​(wit​)=tanh(Wt​⋅wit​)

Wt∈RdwW_t \in \mathbb{R}^{d_w}Wt​∈Rdw​ 是一个权重向量，tanhtanhtanh 是一个 non-linear function。ftf_tft​ 被用于计算一个权重，表示 target 中这个词的重要性。

Experiments

Experiment Settings

word embedding：300-dimension GloVe

OOV words are initialized as zero vectors, and all biases are set to zero.

The dimensions of hidden states and fused embeddings are set to 300.

The dimension of position embeddings is set to 50.

In model training, we set the learning rate to 0.001, the batch size to 64, and dropout rate to 0.5.
