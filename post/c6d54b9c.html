Capsule Network with Interactive Attention for Aspect-Level Sentiment Classiﬁcation

EMNLP 2019 [paper]

Chunning Du, Haifeng Sun, Jingyu Wang∗, Qi Qi, Jianxin Liao, Tong Xu, Ming Liu

BUPT

Introduction

方面级情感分析的主要挑战来自于具有多个 aspects 的句子，对于不同的 aspect 表达了不同的情感，导致了重叠的特征表示。然而，现有的大多数神经网络模型利用静态的池化操作或注意力机制去识别情感词，因此不足以解决重叠的特征。

为了解决这个问题，我们提出了利用胶囊网络通过 EM 路由算法来构建基于向量的特征表示和聚类特征 (cluster features) 。此外，交互式注意力机制在胶囊路由过程中引入以建模 aspect term 与 context 的语义关系。

迭代路由也能够从全局角度对句子编码。迭代路由机制可以被视为一个至上而下的注意力机制，它比标准的注意力机制更加有效是因为它具有全局的视角。标准的注意力机制只考虑句子中上下文信息的一部分，没有考虑到句子的整体含义，这样会引入噪声，降低预测准确率。

Contributions:

  • We apply capsule network to aspect-level sentiment classiﬁcation to tackle the overlapped features by feature clustering. To the best of our knowledge, there is no work that investigates the performance of capsule network in this task.
  • An interactive attention mechanism is introduced in the capsule routing to help model the semantic relationship between aspect term and context.

Input Embedding Layer

IACapsNet 上下文的输入表示包含 word embeddings wnw_nwn​ 和 position embeddings pnp_npn​ 。Aspect term 的输入表示由 word embedding wnaw_n^awna​ 组成。wn,wna∈Rdww_n, w_n^a \in \mathbb{R}^{d_w}wn​,wna​∈Rdw​ ，dwd_wdw​ 是词向量的维度。

考虑到离 aspect 近的上下文词会有更高的情感极性上的影响，我们引入 position embedding 来编码从词 wnw_nwn​ 到 aspect term 的相对距离 rnr_nrn​ 。我们定义 position embedding matrix P∈Rdp×NP \in \mathbb{R}^{d_p \times N}P∈Rdp​×N ，它是随机初始化的，在训练过程中更新。dpd_pdp​ 是 position embedding 的维度，NNN 表示句子的长度。词对应的 position embeddings pnp_npn​ 可以通过用 rnr_nrn​ 查找 position embedding matrix PPP 来获得。

每个上下文词的 input representation 为 word embeddings 和 position embeddings 的拼接：xn=[wn;pn]∈Rdw+dpx_n=[w_n;p_n] \in \mathbb{R}^{d_w+d_p}xn​=[wn​;pn​]∈Rdw​+dp​ 。

经过BiGRU得到的词表征分别为 hnc,hna∈R2×dlh_n^c,h_n^a \in \mathbb{R}^{2 \times d_l}hnc​,hna​∈R2×dl​ ，dld_ldl​ 是隐藏状态的维度。hah^aha 和 hch^chc 分别是 context 和 aspect term 的词表征。

Primary Capsule Layer

基本的胶囊是一组从对于 hnah_n^ahna​ 和 hnch_n^chnc​ 进行卷积操作输出的神经元。因此，胶囊的输出是一个向量，代表同一对象的不同属性。在方面级情感分类任务中，属性可能包含 sentiment 和 aspect term features。

在我们的模型中实现了基于EM的路由方法，除了高维的输出 M 之外，我们的胶囊中还有一个激活概率 aaa ，这类似于标准神经网络中的激活。

Interactive Attention EM Routing

我们已经决定了基础的胶囊层的所有胶囊 ΩLΩ_LΩL​ 的输出，现在决定哪些胶囊 ΩL+1Ω_{L+1}ΩL+1​ 被激活，以及如何将每个激活的低级胶囊分配到激活的高级胶囊。

基于向量的特征在高级胶囊中通过基于 EM 的算法得到聚集，高级胶囊的输出起着高斯 (Gaussians) 的作用，低级胶囊的输出起着数据点 (datapoints) 的作用。

输出胶囊的均值、方差和激活概率以及输入胶囊的分配概率 RRR 通过在 E-step 和 M-step 之间交替进行迭代更新。它也可以看作是一种反向的平行注意机制，它可以根据句子的整体表征来调整低级词的贡献。

此外，为了建立 aspect 和上下文之间的语义关系模型，我们进一步设计了一种基于交互注意的路由机制。
