Multi-Granularity Self-Attention for Neural Machine Translation

EMNLP-2019 [paper]

Introduction

现在的 SOTA 神经网络机器翻译使用深度 multi-head self-attention network ，版那个没有显式地建模短语信息。但是，先前有关统计机器翻译的工作表明，将基本翻译单元从单词扩展到短语已经产生了实质性的改进，这表明有可能通过对短语的显式建模来提高NMT性能。

在本文中，我们提出了 multi-granularity self-attention (MG-SA) ：一个结合了 multi-head self-attention 和短语建模的神经网络。具体地，我们训练几个 attention heads 去关注n-gram或句法形式中的短语。另外，我们利用短语之间的交互来增强结构建模的优势，这是一个经常被提到的 self-attention 的弱点。

Motivation：

一方面，传统的 MH-SA 建模单个词的依赖，这种情况下， query 直接关注所有的词而不考虑输入句子的潜在结构。我们认为 self-attention 可以通过短语模式进一步提升。

另一方面，最近的研究暗示了 attention heads 的数量从4增加到8甚至是16，attention heads 没有被充分利用，所以很难提升翻译性能。在特定的指导下，可以进一步利用几个 attention heads ，以提高性能。我们希望多粒度短语的归纳偏置可以进一步提升 SANs 的性能，同时保持它的简单性和灵活性。

Multi-Granularity Self-Attention

Framework

MG-SA 旨在通过对词和短语的建模来提升 MH-SA 的能力。我们在传统的词级的记忆上引入不同的短语粒度，以生成短语级记忆。

具体地，我们首先在某个 attention head 中通过函数 FhF_hFh​ 将输入层 H\text{H}H 转换成一个短语级记忆 ：

Hg=Fh(H)\text{H}_g = F_h(\text{H}) Hg​=Fh​(H)

Hg\text{H}_gHg​ 是生成的短语级记忆，hhh 表示第 hhh 个head，用于生成特定粒度的短语记忆。FhF_hFh​ 是一个具有自己可训练参数的 representation function 。

然后，我们在短语级记忆 Hg\text{H}_gHg​ 使用注意力:

Qh∈Rn×dh,Kh∈Rp×dh,Vh∈Rp×dh\text{Q}^h \in \mathbb{R}^{n \times d_h}, \text{K}^h \in \mathbb{R}^{p \times d_h}, \text{V}^h \in \mathbb{R}^{p \times d_h}Qh∈Rn×dh​,Kh∈Rp×dh​,Vh∈Rp×dh​ ，ppp 是由短语的粒度决定的 key 和 value 向量的长度。

基于这 single head self-attention ，MG-SA 的最终输出可以表达为：

MG-SA(H)=[O1,...,ON]\text{MG-SA}(\text{H}) = [\text{O}^1,...,\text{O}^N] MG-SA(H)=[O1,...,ON]

NNN 表示 heads 的数量。一个 head 要么进行传统的词级注意，要么进行一定粒度的短语注意。

Multi-Granularity Representation

如上图，不同的 heads 同时对多个粒度的短语进行建模。为了获得多粒度短语表示，我们首先介绍了短语划分 (phrase partition) 和组合策略 (composition strategies) 。然后，我们描述了短语标签监督和短语交互，以进一步增强短语表示的结构建模。

Phrase Partition

我们将整个序列拆分为 N-grams ，没有重叠。若 x=(x1,...,xT)\text{x}=(x_1,...,x_T)x=(x1​,...,xT​) 是一个序列，x\text{x}x 的短语序列可以被表示为 Px=(p1,...,pM),  M=T/nP_{\text{x}} = (p_1,...,p_M), \; M=T/nPx​=(p1​,...,pM​),M=T/n ，pm=(xn×(m−1),...,xn×m),  1≤m≤Mp_m=(x_{n \times (m-1)}, ..., x_{n \times m}), \; 1 \leq m \leq Mpm​=(xn×(m−1)​,...,xn×m​),1≤m≤M ，nnn 是一个超参数，表示短语的长度。

另外，句法信息已经被证明对 SMT 和 NMT 有帮助，我们进一步引入一个句法短语划分，以表示结构良好的结构信息。句法短语通过使用成分解析树将单词组织为嵌套成分。为了获得视图句法中的短语，我们分解了解析树中前K个层的节点，以捕获短语的前K个粒度级别，如图1 (a) 所示。

解析树中某一层的一个短语可以被定义为 pm=(x1,...,xl)p_m=(x^1,...,x^l)pm​=(x1,...,xl) ，lll 是短语的长度，给定的输入 x\text{x}x 的短语序列为 Px=(p1,...,pM)P_{\text{x}} = (p_1,...,p_M)Px​=(p1​,...,pM​) ，MMM 是序列中短语的数量。

Composition Strategies

给定输入序列的短语序列 Px=(p1,...,pM)P_{\text{x}}=(p_1,...,p_M)Px​=(p1​,...,pM​) ，捕获每个短语的内部局部结构和上下文依赖，进一步生成短语表示 QMQ_MQM​ ，我们对短语序列中的每个短语采用短语组合函数：

gm=COM(pm)g_m = \text{COM}(p_m) gm​=COM(pm​)

COMCOMCOM 是具有所有短语共享参数的组成函数，gm∈R1×dhg_m \in \mathbb{R}^{1 \times d_h}gm​∈R1×dh​ 是组成后的短语表示。合成函数一般的选择有 CNNs、RNNs、Self-attention Networks (SANs)。对于 CNNs ，我们近使用一个 Max-pooling 层；对于 RNNs ，我们使用 LSTM 的最后隐藏状态最为短语表示；对于 SANs ，我们使用短语的 max pooling vector 来作为 query ，提取短语内部特征，生成短语表示。然后，输入序列的短语级记忆可以表示为 Gx=(g1,...,gM)\text{G}_{\text{x}}=(g_1,...,g_M)Gx​=(g1​,...,gM​) 。

Phrase Tag Supervision

最近的研究显示 SANs 的 heads 上辅助监督可以进一步提升语义角色标注的性能。在本工作中，我们将标签信息作为句法短语表示的辅助监督。我们认为所提出的框架提供了一种自然的方法来合并短语表示的句法标记信号。

详细地，在短语组成之后给定短语级的记忆 Gx=(g1,...,gM)\text{G}_{\text{x}} = (g_1,...,g_M)Gx​=(g1​,...,gM​) ，我们预测每个短语表示的短语标签。我们抽取成分解析树中每个短语的节点，以生成短语标签序列 Tx=(t1,...,tM)\text{T}_{\text{x}} = (t1, ..., t_M)Tx​=(t1,...,tM​) ，tit_iti​ 表示每个短语的标签。例如，“NP” 是解析树的第二层的短语 “a talk” 的标签。我们使用短语表示来计算短语标签的概率：

pθi=softmax(Wtgi+bt),i=1,...,M,p_{\theta_i} = softmax(W_t g_i + b_t), i=1,...,M, pθi​​=softmax(Wt​gi​+bt​),i=1,...,M,

短语标签损失为：

Ltag=−∑i=1Mtilog pθi(ti).\mathcal{L}_{tag} = -\sum_{i=1}^M t_i log\,p_{\theta_i} (t_i). Ltag​=−i=1∑M​ti​logpθi​​(ti​).

Phrase Interaction

我们引入短语相互作用的方法来更好的建模短语表示之间的依赖。由于循环 (recurrence) 已经被证明对捕获结构信息很重要，我们提出引入 recurrence 来交互短语，进一步建模短语之间的潜在结构。具体地，我们将 recurrence function REC(⋅)REC(·)REC(⋅) 应用到 Gx=(g1,...,gM)\text{G}_{\text{x}} = (g_1, ..., g_M)Gx​=(g1​,...,gM​) ：

Hg=REC(Gx)\text{H}_g = REC(\text{G}_{\text{x}}) Hg​=REC(Gx​)

Hg\text{H}_gHg​ 是输入层 H\text{H}H 最后短语级记忆。对于 REC(⋅)REC(·)REC(⋅) 一个一般的选择是使用 LSTM。最近，Shen et al. (2018) 引入一个新的面向句法的归纳偏置，成为 ordered neurons，使LSTM模型在不破坏其顺序形式的情况下执行类似树的成分，并且提出了一个 LSTM 变体 —— Ordered Neurons LSTM (ON-L STM)。
