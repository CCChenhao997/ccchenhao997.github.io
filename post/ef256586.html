A Human-Like Semantic Cognition Network for Aspect-Level Sentiment Classiﬁcation

AAAI-2019 [paper]

Graduate School at Shenzhen, Tsinghua University

Introduction

本文根据人类阅读认知过程（pre-reading、active reading、post-reading）的原理，提出了一种新颖的 Human-like Semantic Cognition Network (HSCN) 用于ABSA任务。

我们首先设计一个词级的交互感知模块，用于捕获上下文词与给定的目标词 (target words) 之间的联系，这个过程可以视为 pre-reading 。

其次，为了模拟 active reading 的过程，我们提出了一种 target-aware 语义蒸馏模块，以生成用于方面级情感预测的目标特定的上下文表示。

然后，我们进一步设计了一个语义偏差度量模块，以衡量目标特定的上下文表示与给定的目标之间的语义偏差。将所测得的语义偏差用于以反馈调节方式对上述 active reading 过程进行微调。

In the pre-reading stage, humans set the purpose of reading and preview the text with prior knowledge to form the initial reading cognition.

Active reading is a complex cognitive process by which the reader constructs meaning from text.

This post-reading stage (rereading) creates opportunities for deeper understanding and error correction.

Methodology

Overview

word-level interactive perception module

词级交互感知模块预先读取文本内容和给定目标，形成初始认知，捕获上下文中的词与给定目标词之间的相关性。

target-aware semantic distillation module

在初始感知之后，提出了一种目标感知的跳读机制来选择上下文中与目标相关的词。以选定的目标相关词为输入，设计语义合成模块，得到目标特定的上下文表示。

semantic feedback module

该模块测量目标特定的上下文表示和给定目标表示之间的语义偏差。这种语义偏差在一定程度上反映了我们对上下文语义的理解程度。我们使用语义偏差以反馈调节的方式来调整上述主动阅读 (active reading) 过程。

Target-aware Semantic Distillation Module

target-aware semantic distillation 模块包含两个部分：

  • the skip-reading module for selecting target-related words
  • the semantic composition module for encoding the target-speciﬁc context representation

Skip-reading Module
