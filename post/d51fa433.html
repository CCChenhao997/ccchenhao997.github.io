QANet: Combining Local Convolution with Global Self-Attention for Reading Comprehension

ICLR-2018 [paper] [code]

Carnegie Mellon University, Google Brain

Introduction

本文提出了一个新的 QA 模型框架，称为 QANet。Encoder 模块使用了大量的 CNN 和 self-attention ，CNN 可以建模局部的交互，self-attention 可以建模全局的交互。这样的设计比之前基于 RNN 结构的模型要快很多，所有作者使用数据增强（反向翻译）生成了更多的数据来训练模型。

The Model

Problem Formulation

给定一个具有 nnn 个词的上下文段落 C={c1,c2,...,cn}C=\{c_1,c_2,...,c_n\}C={c1​,c2​,...,cn​} 和具有 mmm 个词的 query sentence Q={q1,q2,...,qm}Q=\{q_1,q_2,...,q_m\}Q={q1​,q2​,...,qm​} ，输出一个从原始段落 CCC 中的一个 span S={ci,ci+1,...,ci+j}S=\{c_i,c_{i+1},...,c_{i+j}\}S={ci​,ci+1​,...,ci+j​} 。

接下来，我们使用 xxx 表示任意 x∈C,Qx \in C,Qx∈C,Q 的原始单词及 embedded vector 。

Model Overview

我们的模型主要包含五个部分：embedding layer，embedding encoder layer，context-query attention layer，model encoder layer，output layer。

1. Input Embedding Layer.

对于每个词 www ，我们将它的 word embedding 和 character embedding 进行拼接。Word embedding 在训练过程中是固定的，并且使用 p1=300p_1=300p1​=300 维的预训练 Glove 进行初始化。所有 OOV 的词被随机初始化，且可以被训练。

每个字符都表示为一个维数为 p2=200p_2 = 200p2​=200 的可训练向量，每个词的长度被截断或 paded 到16。我们采取矩阵中每一行的最大值来得到每个词的固定大小的向量表示。

最终，一个词的表示为 [xw;xc]∈Rp1+p2[x_w;x_c] \in \mathcal{R}^{p_1 + p_2}[xw​;xc​]∈Rp1​+p2​ ，xwx_wxw​ 和 xcx_cxc​ 分别为 word embedding 和 character embedding 的卷积输出。

再对其使用一个两层的 highway network 。为了简化，我们在该层的输出也使用 xxx 表示。

2. Embedding Encoder Layer.

The encoder layer 是以下基本模块的堆叠：[convolution-layer × # + self-attention-layer + feed-forward-layer]

我们使用的是 depthwise separable convolutions ，它可以提高内存效率，具有更好的泛化性。The kernel size = 7，the number of filters is d=128d = 128d=128 and the number of conv layers within a block is 4。

对于 self-attention-layer ，我们采用的是 Transformer 中的 multi-head attention mechanism。所有层的 heads 数量为8。

这些每一个基础操作 (conv/self-attention/ffn) 都被放置在一个 residual block 中。对于一个输入 xxx 和一个给定的操作 fff ，输出为 f(layernorm(x))+xf(layernorm(x))+xf(layernorm(x))+x 。

Note that the input of this layer is a vector of dimension p1+p2=500p_1 + p_2 = 500p1​+p2​=500 for each individual word, which is immediately mapped to d = 128 by a one-dimensional convolution. The output of this layer is a also of dimension d = 128.

3. Context-Query Attention Layer.

我们使用 CCC 和 QQQ 分别表示 the encoded context 和 query。

The context-to-query attention 的构建如下：

我们首先计算每对 context 和 query word 之间的相似度，得到一个相似度矩阵 S∈Rn×mS \in \mathbb{R}^{n \times m}S∈Rn×m ，然后我们对其中的每一行使用 softmax 进行归一化，得到一个矩阵 S‾\overline{S}S 。

Context-to-query attention 计算为 A=S‾⋅QT∈Rn×dA=\overline{S}·Q^T \in \mathcal{R}^{n \times d}A=S⋅QT∈Rn×d 。相似度函数使用的是 trilinear 函数：

f(q,c)=Wo[q,c,q⊙c]f(q,c)=W_o[q,c,q \odot c] f(q,c)=Wo​[q,c,q⊙c]

⊙\odot⊙ 是 element-wise multiplication。

我们发现 DCN attention (Xiong et al., 2016) 比简单地应用 context-to-query attention 要提供一点帮助，所以我们采取这种策略。具体地，我们用 softmax 计算 SSS 的列归一化矩阵 S‾‾\overline{\overline{S}}S ，the query-to-context attention 为 B=S‾⋅S‾‾T⋅CTB=\overline{S}·\overline{\overline{S}}^T·C^TB=S⋅ST⋅CT 。

4. Model Encoder Layer.

该层每个位置的输出为 [c,a,c⊙a,c⊙b][c,a,c \odot a, c \odot b][c,a,c⊙a,c⊙b] ，aaa 和 bbb 分别是 attention matrix AAA 和 BBB 的行向量。该层的参数与 Embedding Encoder Layer 一样，除了卷积层的数量为2。

5. Output layer.

This layer is task-speciﬁc. Each example in SQuAD is labeled with a span in the context containing the answer.

我们采用 Seo et al. (2016) 中的策略去预测上下文中每个位置是 answer 的开始位置或结束位置的概率。具体地，开始位置和结束位置的概率被建模为：

p1=softmax(W1[M0;M1]),  p2=softmax(W2[M0;M2])p^1 = softmax(W_1[M_0;M_1]),\; p^2=softmax(W_2[M_0;M_2]) p1=softmax(W1​[M0​;M1​]),p2=softmax(W2​[M0​;M2​])

M0,M1,M2M_0,M_1,M_2M0​,M1​,M2​ 分别是三个 model encoders 的输出。一个 span 的得分为它的开始位置概率和结束位置概率的乘积。

Finally, the objective function is deﬁned as the negative sum of the log probabilities of the predicted distributions indexed by true start and end indices, averaged over all the training examples:

L(θ)=−1N∑iN[log(pyi11)+log(pyi22)]L(\theta) = - \frac{1}{N} \sum_i^N [log(p^1_{y^1_i})+log(p^2_{y^2_i})] L(θ)=−N1​i∑N​[log(pyi1​1​)+log(pyi2​2​)]

yi1y^1_iyi1​ 和 yi2y^2_iyi2​ 分别为样本 iii 的真实开始位置和结束位置。θ\thetaθ 包含所有的可训练变量。
