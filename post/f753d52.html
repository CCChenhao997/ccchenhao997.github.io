小程序及代码结构简介
2020-03-21|微信小程序
字数总计: 751|阅读时长: 2 分钟

小程序的特点：

  • 触手可及
  • 不占内存
  • 无需安装卸载

代码结构
  • app.js 注册微信小程序应用；
  • app.json 进行微信小程序的全局配置，包括网络请求的超时时间、一些窗口的表现、各个页面的注册的路径；
  • app.wxss 设置微信小程序全局的样式；
  • pages 里面存放的是微信小程序所有的小程序页面，每个小程序页面最多由四个文件组成，分别是js文件、json文件、wxml文件和wxss文件。
    • js文件帮我们去处理微信小程序页面的逻辑和一些数据交互，
    • json文件是帮我们去配置微信小程序页面的配置信息，
    • wxml文件是用来帮我们展示小程序页面的一些元素和内容，
    • wxss文件是帮我们去设置小程序页面这些元素的样式。
  • utils.js 存放一些工具函数，可以达到我们代码复用的目的；
  • project.config.json 保存微信开发者工具的配置信息。

调试器

调试器模块各个面板的作用：

  • Console 打印小程序页面的一些调试log
  • Source 会列出微信小程序页面的所有脚本文件，我们可以对这些脚本文件进行一个断点调试
  • Network 展示各个网络请求的状态信息以及请求资源的响应数据等
  • Storage 当我们在小程序里面通过调用 wx.setStorage 和 wx.setStorageSync 函数设置缓存时，我们可以动态地去修改这些缓存数据
  • AppData 时微信小程序页面上真实展示的数据，我们也可以在这里动态地修改来查看小程序页面在模拟器上不同设备的兼容性的情况
  • Wxml 展示了微信小程序页面的各个组件元素，以及我们可以通过对这些组件元素对它进行样式属性的修改
  • Sensor 地理位置信息和设备旋转角度的展示，可以动态地去修改地理位置信息来模拟小程序页面在不同的地理位置下的表现

rpx与px

rpx与px的区别：

rpx时一个响应式的屏幕像素，rpx单位是微信小程序中css的尺寸单位，rpx可以根据屏幕宽度进行自适应。规定屏幕宽为750rpx。如在 iPhone6 上，屏幕宽度为375px，共有750个物理像素，则750rpx = 375px = 750物理像素，1rpx=0.5px=1物理像素。
