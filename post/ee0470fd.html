Introduction

There exist seven subtasks in ABSA:

The following divergences make it difï¬cult to solve all subtasks in a uniï¬ed framework.

  1. Input: Some subtasks ( AE, OE, AESC, Pair and Triplet) only take the text sentence as input, while the remained subtasks ( ALSC and AOE) take the text and a given aspect term as input.
  2. Output: Some tasks (AE, OE, ALSC, AOE) only output a certain type from a, s or o, while the remained tasks (AESC, Pair and Triplet) return compound output as the combination of aaa, sss and ooo.
  3. Task Type: There are two kinds of tasks: extraction task (extracting aspect and opinion) and classiï¬cation task (predicting sentiment).

Motivated by the above observations, we proposes a uniï¬ed generative framework to address all the ABSA subtasks.

Speciï¬cally, we model the extraction and classiï¬cation tasks as the pointer indexes and class indexes generation, respectively.

Contributions:

  • We formulate both the extraction task and classiï¬cation task of ABSA into a uniï¬ed index generation problem. Unlike previous uniï¬ed models, our method needs not to design speciï¬c decoders for different output types.
  • With our re-formulation, all ABSA subtasks can be solved in sequence-to-sequence framework, which is easy-to-implement and can be built on the pre-trained models, such as BART.
  • We conduct extensive experiments on four public datasets, and each dataset contains a subset of all ABSA subtasks. To the best of our knowledge, it is the ï¬rst work to evaluate a model on all ABSA tasks.
  • The experimental results show that our proposed framework signiï¬cantly outperforms recent SOTA methods.
