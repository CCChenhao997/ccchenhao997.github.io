Introduction

RNN + Attention 很适合 ABSA 任务，而且也去得了很好的效果。但是，之前的注意力机制可能会关注到不相关的词，因此会损失性能，特别是在具有多个 aspects 的又长又复杂的句子。

在本文中，我们提出了一种新架构，称为 Hierarchical Gate Memory Network (HGMN) 。首先，我们使用 hierarchical gate mechanism 去学习选择与给定的 aspect 相关的部分。之后，我们在 aspect-specific memory 上使用 CNN。
