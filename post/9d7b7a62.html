2020-07-07|NLP

Text Level Graph Neural Network for Text Classiﬁcation

ENMLP-2019 [paper]

Lianzhe Huang, Dehong Ma, Sujian Li, Xiaodong Zhang and Houfeng WANG

MOE Key Lab of Computational Linguistics, Peking University

Introduction

之前基于GNN的方法主要面对的是固定语料库级别图结构的实际问题，这些问题不支持在线测试和高内存消耗。为了解决这一问题，我们提出了一种新的基于GNN的模型，它为每个输入文本建立全局参数共享的图，而不是整个语料库的一个单独的图。该方法消除了单个文本和支持在线测试的整个语料库之间的依赖性负担，但仍然保留了全局信息。此外，我们在文本中使用更小的窗口来构建图，这样不仅可以提取更多的局部特征，而且可以显著减少边数和内存消耗。

Method

首先，我们展示如何根据一个给定的文本构建一个文本级的图，文本级图的所有参数都取自全局共享矩阵。然后，我们在这些图上引入消息传递机制，从上下文中获取信息。最后，我们描述了如何根据学习到的表示来预测给定文本的标签。

Building Text Graph

NNN 和 EEE 是图的节点集合和边集合，NNN 中的词表示和 EEE 中的边权重取自全局共享矩阵。ppp 表示与图中每个单词相连的相邻单词的数量。此外，我们将训练集中出现次数少于 kkk 次的边统一映射到“public”边，以使参数得到充分训练。

与以往的构造图方法相比，我们的方法可以极大地减少图的节点和边的规模。这意味着文本级图可以消耗更少的 GPU 内存。另外，他们的方法对新出现的文本不友好，而我们的方法可以解决这个问题，因为每个文本的图只依赖于其内容。

Message Passing Mechanism

卷积可以从局部特征抽取信息。在图领域，卷积通过谱方法或非谱方法来执行。本文采用了一种非谱方法，即消息传递机制（MPM）进行卷积。MPM 首先从相邻节点收集信息，并根据其原始表示和收集的信息更新其表示，定义为：

节点 nnn 从邻居节点接受消息 (message) Mn∈Rd\text{M}_n \in \mathbb{R}^dMn​∈Rd ，max是归约函数，它将每个维上的最大值组合在一起以形成新的向量作为输出。Nnp\mathcal{N}_n^pNnp​ 节点表示 nnn 的原始文本中最近的 ppp 个词；ean∈R1e_{an} \in \mathbb{R}^1ean​∈R1 是从 aaa 到 nnn 边的权重，可以在训练中更新；rn∈Rdr_n \in \mathbb{R}^drn​∈Rd 表示节点 nnn 之前的表示。ηn∈R1\eta_n \in \mathbb{R}^1ηn​∈R1 是节点 nnn 一个可训练的变量，表示 rnr_nrn​ 的多少信息应该保留。rn′r_n'rn′​ 表示节点 nnn 个更新表示。

MPM使节点的表示受到邻域的影响，这意味着表示可以从上下文中获取信息。因此，即使对于多义词，其在上下文中的确切含义也可以通过来自邻居的加权信息的影响来确定。此外，文本级图的参数取自全局共享矩阵，这意味着表示也可以像其他基于图的模型一样带入全局信息。
