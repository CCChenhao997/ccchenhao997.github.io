Introduction

通过一个具体的例子——基于神经网络的机器翻译——来介绍注意力机制。

传统的方法是 encoder-decoder 结构，使用RNN来实现。虽然相较于纯统计学习方法获得了更好的表现，但是基于RNN结构的有两大缺陷：

  1. RNN具有遗忘性，意味着旧的信息在经过多个时间步之后会被冲刷掉。
  2. 其次，在解码期间没有明确的字对齐，因此焦点分散在整个序列中（即对待每个词的重要性都是一样的）。

左边是传统的 encoder-decoder 架构，右边是基于注意力机制的架构。

使用了注意力机制后的好处：

  • 解决了RNN的记忆性问题
  • 实现了软对齐

本文将从以下几点做综述：

  1. 详细解释注意力机制的基本形式；（Formulation）
  2. 根据特定的任务讨论不同的变体；（Variation）
  3. 探索注意力如何与机器学习中的其他概念或技术（例如预训练和集成）相关联；（Application）
  4. 检查评估注意力表现的方法。（Evaluation）

Variation

在此之前，我们讨论了注意机制的基本形式。由于它的简单性和可解释性，它被广泛应用于各种NLP任务中。然而，这样的注意机制一般不足以应付更复杂的任务。

例如：

Sam walks into the kitchen. (1)

Sam picks up an apple. (2)

Sam walks into the bedroom. (3)

Sam drops the apple. (4)

Q: Where is the apple?

A: Bedroom

潜在的困难在于Q和（3）之间没有直接的关系，因此我们需要设计一个更复杂的机制，以使用上下文中的潜在线索（在本例中是时间推理）来正确定位注意力。

由于近年来人们提出了注意力机制的不同变体，我们将其归纳为几个最具代表性的类别：

Multi-dimensional Attention

基础形式的注意力机制通过对一个序列 V={vi}V=\{v_i\}V={vi​} 中的每一项计算一个标量的数值 αi\alpha_iαi​ ，这可以称为一维的注意力或者向量注意力，因为将所有输出的 α={αi}\alpha = \{\alpha_i\}α={αi​} 拼接到一起是一个向量 α∈Rn\alpha \in R^nα∈Rn 。

Hierarchical Attention

以文档分类为例，对于文本数据的嵌套结构：字符∈单词∈句子∈文档，每一层级都能够对于分类提供不同的信息，则可以相应地构造一个层次注意，即自下而上（即单词到句子）或自上而下（单词到字符）来识别线索或从全局和局部提取重要信息。

自下而上的结构已用于文件分类（(Yang et al. 2016）。两个BiGRUs分别用于生成单词级和句子级上下文表示。然后应用一对层次注意层来获得单词级和句子级编码：

hi(t)h^{(t)}_ihi(t)​ 和 hih_ihi​ 分别表示单词级和句子级的隐藏层表示，uwTu^T_wuwT​ 和 usTu^T_susT​ 分别是单词级和句子级的 pattern vector，且是在训练中是可学习的。最终的句子级表示 ccc 送入逻辑回归层去预测类别。

另一种类型的 hierarchical attention 采用自上而下的方法，例如语法错误纠正。

Consider a corrupted sentence: I have no enough previleges. The idea is to ﬁrst bulid an encoder-decoder architecture similar to the one for machine translation, then apply a word-level attention for global grammar and ﬂuency error correction (I have no enough → I don’t have enough), and optionally a character-level attention for local spelling error correction (previleges → privileges).

自上而下的技术也用于相册摘要中，其中照片级别(photo-level)的注意力用于从相册中选择合适的照片，单词级别的注意力与序列模型集成以生成文本 。

Self Attention

之前介绍的基础形式的Attention也可以被称为外部的注意力，因为我们计算 attention score αi=a(u,vi)\alpha_i = a(u, v_i)αi​=a(u,vi​) ，其中 uuu 是一个外部的 pattern vector。相反，在 self-attention 中，使用序列中的一部分代替外部 pattern uuu，因此也被称为 internal attention。

举一个例子来说明这一点：Volleyball match is in progress between ladies，这里 match 是所有其他标记依赖的句首，理想情况下，我们希望使用self-attention自动捕获这种内在依赖。或者，我们可以将 self-attention 解释为匹配每个元素 viv_ivi​ 与内部的模式向量 v′v'v′：

ei=a(v′,vi)e_i = a(v', v_i) ei​=a(v′,vi​)

Self-attention 的另一动机与词嵌入有关。 具体来说，我们想利用 self-attention 模型以自适应方式学习复杂的上下文 token 表示。 我们可以通过一个词义消歧的例子来说明这一点：

I arrived at the bank after crossing the street.

I arrived at the bank after crossing the river.

这两句中的"bank"的含义是不一样的，第一句中的"bank"意思是银行，第二句的"bank"意思是岸边。

Transformer是一种新型的基于注意力机制的结构，它主要是由前向神经网络和 self-attention 层组成，与之前的 self-attention 模式类似。Transformer中注意力机制的其实是 multi-head self-attention 具体的可见之前的 Attention Is All You Need。

Reusability

在一些问答任务中，一个基本的困难是答案与问题间接相关，因此无法通过基本的注意力技巧轻松解决。然而，如果我们可以通过迭代内存更新（也称为多跳 multi-hop）来逐步引导注意力到答案的正确位置来模拟时间推理过程，就可以实现这一点。直观地说，在每次迭代中，查询都会更新为新内容，而更新后的查询则用于检索相关内容。

q(t+1)=q(t)+c(t)q^{(t+1)} = q^{(t)} + c^{(t)} q(t+1)=q(t)+c(t)

更复杂的更新方法包括在多个时间步长的查询和内容之间构建一个循环网络，或根据内容和位置信息进行输出。结果表明，当给出复杂的时间推理任务时，基于记忆的注意模型可以在几次跳跃后成功地找到答案。

Flexibility

由于键和值被清楚地表示，我们可以自由地将先验知识纳入设计单独的键和值嵌入中，以使它们分别更好地捕获相关信息。具体来说，可以手动设计 key embeddings 来匹配question，而 value embeddings 来匹配响应(response)。在 key-value 记忆网络中，提出了一种窗口级表示形式，以便将 key 构造为围绕 entity tokens 为中心的窗口，而 value 则是那些对应的 entities，旨在实现更有效和准确的匹配。如在 Figure 3 中，“apple” 和 “bedroom” 是 value embeddings，而在它们周围的 tokens 是 key embeddings。

更复杂的结构包括动态记忆网络，整体的结构细分为四个部分：question module、input module、episodic memory module 和 answer module。这种模块化的设计可以实现领域知识的逐段注入、模块之间的有效通信以及对传统问题回答之外的更广泛任务的泛化。

一种相似的结构被提出用于处理文本和视觉问答任务，其中视觉输入被送入一个深卷积网络中，高层特征被提取并处理成一个注意力网络的输入序列。

如果我们进一步将 memory 和 query 表示扩展到问答任务之外的领域，基于记忆的注意力技术也被用于方面和观点挖掘，query表示为 aspect prototype；在推荐系统中，用户称为 memory component，items 成为 queries；在主题模型中，从深层网络中提取的潜在主题表示构成了 memory 等。
