Sequential Learning of Convolutional Features for Effective Text Classiﬁcation
2020-05-10|NLP
字数总计: 1.6k|阅读时长: 5 分钟

Sequential Learning of Convolutional Features for Effective Text Classiﬁcation

EMNLP-2019 [paper]

Avinash Madasu, Vijjini Anvesh Rao

Samsung R&D Institute, Bangalore

Introduction

本文是通过分析 CNN 的 convolution ﬁlters 和 max pooling 在文本分类任务上的不足，然后提出了 Sequential Convolutional Attentive Recurrent Network (SCARN) 。并且效果比其他的 recurrent convolutional architectures 效果好。

流行的 CNN 结构在文本上使用卷积是通过一个固定的 window size，这样的操作是否能保存词之间的序列信息？如果对词序列进行 random shuffled 并对其进行卷积，会有什么影响？在 LSTM 中，random shuffling 会导致很差的性能，random ordering 损害了序列学习。然而，在一个固定的 window size 上卷积，并没有强有力的证据证明它保存了序列信息。

之前提出的 CNN 结构都是在卷积输出后使用 max pooling 操作，来捕获最重要的特征。这引发了另外一个问题，被 max pooling 选择的特征是否总是输入的最重要的特征？

Contribution:

  • We propose a new recurrent convolutional model for text classiﬁcation by discussing the shortcomings of max pooling operation and also the strength and weakness of convolution operation.
  • We evaluate the proposed model’s performance on seven benchmark text classiﬁcation datasets. The results show that the proposed model achieves better results with lesser number of parameters than other recurrent convolutional architectures.

Understanding Convolution and Max pooling

Convolution operation

What it doesn’t learn

卷积操作可以被解释为一个沿着时间步做加权求和的操作。这可能会导致序列信息的损失。为了证明我们的假设，我们做了一个实验来验证，在实验中，我们训练一个 CNN ，它的 window size 从 1 到句子最大长度变化。我们再将句子中所有的词进行 randomly shuffling，如 “read book forget movie” to “book read movie forget”。实验结果如下图：

我们从实验中观察到：

  • 由于随机排序的性能和正确排序之间的差距很小，因此 CNN 无法完全学到顺序信息。图中 correct ordering on window size 7 和 random ordering on window size 1 的性能差不多。
  • 随着 window size 的增加，正确和不正确排序之间的差异会减小，最终当 window size 达到完整句子长度时收敛。这表明 CNNs 捕获序列信息的能力随着 window size 的增加而下降。

What it learns

为了理解 convolution ﬁlters 做了什么，我们在 SST2 数据集上训练一个单窗口大小的 CNN。由于卷积作用于一个词上时，它不能捕获序列信息。因此，不管输入句子的词序如何，词总是有相同的卷积输出。本质上，这里的卷积层就像一个 embedding transformation，输入 embedding 空间被转换成另一个空间。为了理解我们使用 t-SNE (一种降维方法) 映射各个 embeddings ：

  • 绿色词在原始输出和卷积输出之间的转换，表明卷积层能够调整它们的输入 embedding，使它们更接近于积极情感簇，而不是原始语义簇。因此，在这种转换后的 embeddings 上进行序列学习将比原始语义 embedding 更有效。
  • 有些词例如“pretty”可能语义上更接近正向情感的词。但是在该数据集中，我们发现 “pretty good” 和 “pretty bad” 这种短语经常出现。因此，经过转换后的 “pretty” 的位置在正向簇和负向簇之间。也就是说，卷积层能够鲁棒地创建一个可以处理任何任务特定噪声的转换。

以上的实验观察显示了卷积操作的有效性。这是因为 single convolution ﬁlter 的输出值捕获了原始词嵌入的所有特征的加权求和，使网络学习到更多的任务相关的特征。

考虑到以上几点，我们观察到卷积操作不能充分利用序列信息，特别是在较大的窗口大小上。然而，我们确实看到了它们在使输入 embeddings 符合对任务有特定意义的表示的有效性。因此，对于序列学习不能依靠卷积，在我们提出的 SCARN model 中使用了循环结构。

在第一个子网络中，使用单窗口大小的卷积对输入 III 进行操作。卷积 filter 从输入中学习到高级表示。接着，卷积的输出送入 LSTM 网络中，进行序列地训练。

在第二个子网络中，输入 III 直接送入 LSTM 中，然后应用一个注意力机制。
