Bidirectional Attention Flow for Machine Comprehension

ICLR 2017 [paper]

Minjoon Seo 1∗ Aniruddha Kembhavi 2 Ali Farhadi 1,2 Hananneh Hajishirzi 1

University of Washington 1 , Allen Institute for Artiﬁcial Intelligence 2

Introducation

近年来，注意机制已经成功地应用到MC。通常，这些方法使用注意力集中在上下文的一小部分，并用固定大小的向量对其进行汇总 (summarize)，在时间上耦合注意力，并且/或者通常形成单向注意力。

Recently, attention mechanisms have been successfully extended to MC. Typically these methods use attention to focus on a small portion of the context and summarize it with a ﬁxed-size vector, couple attentions temporally, and/or often form a uni-directional attention.

在本文中，我们介绍了双向注意力流 (BIDAF) 网络，它是一个多阶段的分层过程，用于表示不同粒度级别的上下文，并使用双向注意流机制来获得无需提前汇总的查询感知上下文表示。

In this paper we introduce the Bi-Directional Attention Flow (B I DAF) network, a multi-stage hierarchical process that represents the context at different levels of granularity and uses bidirectional attention ﬂow mechanism to obtain a query-aware context representation without early summarization.

以往研究中的注意机制通常具有以下一个或多个特征。

首先，计算出的注意权重通常用于从上下文中提取最相关的信息，以便通过将上下文归纳为固定大小的向量来回答问题。

First, the computed attention weights are often used to extract the most relevant information from the context for answering the question by summarizing the context into a ﬁxed-size vector.

其次，在文本域中，它们通常在时间上是动态的，因此当前时间步的注意力权重是前一个时间步的参与向量的函数。

Second, in the text domain, they are often temporally dynamic, whereby the attention weights at the current time step are a function of the attended vector at the previous time step.

第三，它们通常是单向的，其中查询关注上下文段落或图像。

Third, they are usually uni-directional, wherein the query attends on the context paragraph or the image.

本文提出的 BiDAF 包括字符级、词级和上下文级的embeddings，并使用双向注意流来获得查询感知上下文表示。该注意机制对以前流行的注意力范式进行了以下改进。

首先，我们的注意力层不用于将上下文段落汇总为固定大小的向量。 取而代之的是，在每个时间步上计算注意力，并允许每个时间步的参与向量以及之前层的表示流入后继的建模层。这减少了由提前汇总引起的信息丢失。

First, our attention layer is not used to summarize the context paragraph into a ﬁxed-size vector. Instead, the attention is computed for every time step, and the attended vector at each time step, along with the representations from previous layers, is allowed to ﬂow through to the subsequent modeling layer. This reduces the information loss caused by early summarization.

其次，我们使用无记忆注意机制。也就是说，当我们通过时间上迭代地计算注意力，每个时间步的注意只是当前时间步的查询和上下文段落的函数，不直接依赖于上一时间步的注意。

Second, we use a memory-less attention mechanism. That is, while we iteratively compute attention through time as in Bahdanau et al. (2015), the attention at each time step is a function of only the query and the context paragraph at the current time step and does not directly depend on the attention at the previous time step.

我们假设这种简化会导致注意力层 (attention layer) 和建模层 (modeling layer) 之间的分工。它迫使注意力层专注于学习查询和上下文之间的注意 (attention)，并使建模层专注于学习查询感知上下文表示（注意力层的输出）中的交互。它还允许每个时间步的注意力不受之前时间步的错误关注的影响。

It forces the attention layer to focus on learning the attention between the query and the context, and enables the modeling layer to focus on learning the interaction within the query-aware context representation (the output of the attention layer). It also allows the attention at each time step to be unaffected from incorrect attendances at previous time steps.

第三，我们在两个方向上使用注意机制，即查询到上下文和上下文到查询，这两种机制相互提供了互补的信息。

Third, we use attention mechanisms in both directions, query-to-context and context-to-query, which provide complimentary information to each other.

Model

我们的机器理解模型是一个分层的多阶段过程，由六层组成：

  1. Character Embedding Layer maps each word to a vector space using character-level CNNs.
  2. Word Embedding Layer maps each word to a vector space using a pre-trained word embedding model.
  3. Contextual Embedding Layer utilizes contextual cues from surrounding words to reﬁne the embedding of the words. These ﬁrst three layers are applied to both the query and context.
  4. Attention Flow Layer couples the query and context vectors and produces a set of queryaware feature vectors for each word in the context.
  5. Modeling Layer employs a Recurrent Neural Network to scan the context.
  6. Output Layer provides an answer to the query.
