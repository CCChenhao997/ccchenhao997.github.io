lambda表达式

格式 lambda 参数列表:函数体

三元运算符

三元运算符通常在Python里被称为条件表达式，这些表达式基于真(true)/假(false)的条件判断。

map函数

map() 会根据提供的函数对指定序列做映射。
格式map(func, *iterables) --> map object

filter 过滤器

格式filter(function or None, iterable) --> filter object

reduce函数

reduce() 函数会对参数序列中元素进行累积。
