CAN: Constrained Attention Networks for Multi-Aspect Sentiment Analysis

EMNLP-2019 [paper]

Nankai University, IBM Research - China

Introduction

注意力机制会引入噪声，导致降低性能，所以本文提出 constrained attention networks (CAN) 。在一个句子中，如果出现多个不同的 aspect ，它们对词的注意力分布应该相互正交 (orthogonal) 。对于每个 aspect 仅有少量相关词表达观点，所以每个 aspect 的观点表达分布是稀疏的 (sparse) 。

正交正则化倾向于使多个 aspects 的注意权重注重于句子的不同部分，更少的重叠。稀疏正则使每个 aspect 的注意力权重仅注重于少量的词。

此外，本文还引入了一个辅助任务 aspect category detection (ACD) ，两个任务受益于共享上下文。

Model

数据集中预定义的 aspect categories A=A1,...,ANA = {A_1,...,A_N}A=A1​,...,AN​ 。给定一个句子 S={w1,w2,...,wL}S=\{w_1,w_2,...,w_L\}S={w1​,w2​,...,wL​} ，包含 KKK 个 aspects As={A1s,...,AKs},As⊆AA^s = \{A_1^s,...,A_K^s\}, A^s \subseteq AAs={A1s​,...,AKs​},As⊆A 。

ALSC 任务预测每个 aspect Aks∈AsA_k^s \in A^sAks​∈As 的情感极性，辅助任务 ACD 检查每个 aspect An∈AA_n \in AAn​∈A 是否在句子中提到了。

Embedding and LSTM Layers

传统的 single-aspect 情感分析单独处理每个 aspect 。在这种设置下，一个具有 KKK 个aspects的句子 SSS 将被复制 KKK 个样本。例如，一个句子 SSS 包含两个 aspects ：A1sA_1^sA1s​ 极性为 p1p_1p1​ ，A2sA_2^sA2s​ 极性为 p2p_2p2​ 。两个样本将被构建：<S,A1s,p1><S,A_1^s,p_1><S,A1s​,p1​> 和 <S,A2s,p2><S,A_2^s,p_2><S,A2s​,p2​> 。本文中 multi-aspect 情感分析方法一起处理多个方面，用一个样本作为输入：<S,[A1s,A2s],[p1,p2]><S,[A_1^s,A_2^s],[p_1,p_2]><S,[A1s​,A2s​],[p1​,p2​]> 。

输入的句子 {w1,w2,...,wL}\{w_1,w_2,...,w_L\}{w1​,w2​,...,wL​} 首先转换为一个向量序列 {v1,v2,...,vL}\{v_1,v_2,...,v_L\}{v1​,v2​,...,vL​} ，句子中 KKK 个 aspects 转换为向量 {u1s,...,uKs}\{u_1^s,...,u_K^s\}{u1s​,...,uKs​} ，aspect categories {u1,...,uN}\{u_1,...,u_N\}{u1​,...,uN​} 。

句子的 word embeddings 输入到 LSTM 网络中，输出隐藏状态 H={h1,h2,...,hL}H=\{h_1,h_2,...,h_L\}H={h1​,h2​,...,hL​} ，embedding 和 hidden state 的size都设置为 ddd 。

Task-Speciﬁc Attention Layer

ALSC 和 ACD 任务共享 LSTM 层输出的隐藏状态，但它们的注意力权重计算是单独的。注意力权重用于计算 aspect-specific 句子表示。

ALSC Attention Layer

给定一个包含 KKK 个 aspects 的句子 SSS ，As={A1s,...,AKs}A^s=\{A_1^s,...,A_K^s\}As={A1s​,...,AKs​} ，对于每个 aspect AksA_k^sAks​ ，它的注意力权重计算如下：

ACD Attention Layer

我们将 ACD 任务视为针对 NNN 个 aspect categories 的集合的多标签分类问题。

对于每个 aspect An∈AA_n \in AAn​∈A ，它的注意力权重计算为：

βn=softmax(zbTtanh(W1bH+W2b(un⊗eL)))\beta_n = softmax({z^b}^T tanh(W_1^bH + W_2^b(u_n \otimes e_L))) βn​=softmax(zbTtanh(W1b​H+W2b​(un​⊗eL​)))

使用分离参数的原因是，就同一 aspect 而言，ALSC的注意力更多地集中在观点词上，而ACD的注意力更多地集中在方面目标词上。

Regularization Layer

我们通过增加它们的注意力权重的约束，以同时处理多个 aspects 。

注意，该层仅用于训练阶段，因为 groud-truth aspects 是已知的，用于计算正则损失，影响参数在反向传播中的更新。

Task-Speciﬁc Prediction Layer
