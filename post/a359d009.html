Syntax-Aware Aspect Level Sentiment Classiﬁcation with Graph Attention Networks

EMNLP-2019 [paper]

Binxuan Huang, Kathleen M. Carley

School of Computer Science Carnegie Mellon University

Introduction

之前的基于神经网络的方法很大程度上忽略了一个句子中的句法结构。在本文中，我们提出了一个新型的 target-dependent graph attention network (TD-GAT) 用于方面级情感分析，显式的利用了词之间的依赖关系。利用 dependency graph，它直接从方面目标的句法上下文传播情感特征。

我们采用多层图注意网络将重要句法邻域词的情感特征传播到方面目标。我们进一步在TD-GAT中加入一个LSTM单元，以便在递归邻域扩展过程中跨层显式地捕获方面相关信息。

Method

Text Representation

根据一个长度为 nnn 的句子 s=[w1,w2,...,wi,...,wn]s=[w_1,w_2,...,w_i,...,w_n]s=[w1​,w2​,...,wi​,...,wn​] 和一个 aspect target wiw_iwi​ ，我们首先将每个词映射到一个低维的 word embedding vector。对于每个词 wiw_iwi​ ，我们得到一个 vector xi∈Rdx_i \in R^dxi​∈Rd ，ddd 是 word embedding space 的维度。

我们使用现成的依赖解析器将句子转换成依赖图。每个节点表示一个词，并与一个 embedding vector 相关联，作为其局部特征向量。两个词之间的无向边意味着两个词是句法相关的。在下图中，我们展示了句子 “delivery was early too” 的 dependency graph，对于 target “delivery” ，我们可以将特征从其 2-hop 邻域传播到 1-hop 邻域，然后再传播到自身。

对于一个具有多个词的 aspect target，我们首先用一个特殊的符号 “_target_ ” 来代替整个 target word sequence，然后将修改的句子送入 dependency parser。结果，在 dependency graph 中存在一个表示 target sequence 的 meta-node ，其局部特征向量是 target 中的词的 word embedding 的平均化。
