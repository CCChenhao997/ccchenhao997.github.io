Introduction

最近，图神经网络被研究融合句法依赖用于该任务，并且取得了SOTA结果。然而，现存的方法没有考虑依赖标签信息，这在直观上可能是有用的。为了解决这个问题，我们研究了一种新型的关系图注意网络，它集成了类型化的句法依赖信息 (typed syntactic dependency information) 。

从图1(b)和图1©可以看出，虽然“Arsenal”在两句话中都与“defeated”有相似的依赖弧关系，但情感极性不同。显然，区分标签类型可以帮助区分这种情况。一方面，考虑这样的信息可以使句法结构更加稀疏，从而增加学习的难度。因此，这仍然是一个开放的研究问题：如何有效地利用这种细粒度的句法特征进行更好的 targeted 情感分类。

我们研究了一个图注意网络来集成这样的类型化的依赖特征。特别地，提出的模型融合了标签特征到注意力机制中，使用扩展的注意力函数来指导信息从目标提及的句法上下文传播到目标提及本身。我们将模型命名为 relational graph attention network (RGAT) ，借助这些标签特征，我们的模型可以更好地捕获到词之间的关系，因此可以处理 Figure 1 中的问题。此外，依赖标签可以作为额外的特征，用于丰富词的表示。

Approach

对于我们的模型，每个样本由三个部分组成：一个 target、一个句子和一个句子的依赖树。我们将这三个部分表示成一个三元组：(S,Sτ,T)(S, S^{\tau}, T)(S,Sτ,T)，其中 S={w1,w2,...wi,...,wi+m,...wn}S=\{w_1,w_2,...w_i,...,w_{i+m},...w_n\}S={w1​,w2​,...wi​,...,wi+m​,...wn​} 表示输入的句子，Sτ={wi,wi+1,...,wi+m−1}S^{\tau}=\{w_i,w_{i+1},...,w_{i+m-1}\}Sτ={wi​,wi+1​,...,wi+m−1​} 表示 target，TTT 表示关于 SSS 的依赖树。SSS 和 SτS^{\tau}Sτ 的长度分别为 nnn 和 mmm 。依赖树可以表示成一个图 T=(V,A,R)T=(V,A,R)T=(V,A,R) ，VVV 包含所有的词 {w1,w2,...,wn}\{w_1,w_2,...,w_n\}{w1​,w2​,...,wn​} ，AAA 是一个邻接矩阵 A∈Rn×nA \in R^{n \times n}A∈Rn×n ，如果词 wiw_iwi​ 与 wjw_jwj​ 之间存在依赖弧，则 Aij=1A_{ij}=1Aij​=1，否则 Aij=0A_{ij}=0Aij​=0 。 RRR 是一个关系矩阵，RijR_{ij}Rij​ 对应于 AijA_{ij}Aij​ 的标签。

Input Layer

我们考虑两种类型的 word embeddings：GloVe embeddings 和 BERT embeddings 。

Glove Embeddings

根据一个句子 S={w1,w2,...,wn}S=\{w_1,w_2,...,w_n\}S={w1​,w2​,...,wn​} ，我们对 GloVe word embedding matrix 执行一个查找操作，对于每个词 wiw_iwi​ 获得一个向量 vi∈Rdwv_i \in \mathcal{R}^{d_w}vi​∈Rdw​ ，dwd_wdw​ 是 word embeddings 的维度。

与之前的工作一样，我们也是用了 POS-tag embeddings ti∈Rdtt_i \in R^{d_t}ti​∈Rdt​ 和 position embeddings pi∈Rdpp_i \in R^{d_p}pi​∈Rdp​ 作为额外输入。因此，一个词 wiw_iwi​ 的最终表示为 xi=[vi;ti;pi]x_i=[v_i;t_i;p_i]xi​=[vi​;ti​;pi​] 。

为了捕获每个词的丰富的特征，我们使用 BiLSTM 层来建模双向的上下文信息。根据一个 word embedding sequence x={x1,x2,...,xn}x=\{x_1,x_2,...,x_n\}x={x1​,x2​,...,xn​} ，经过 BiLSTM 输出隐藏状态 {h10,h20,...,hn0}\{h_1^0,h_2^0,...,h_n^0\}{h10​,h20​,...,hn0​} 。

BERT Embeddings

我们采用 BERT 来对每个词 wiw_iwi​ 提供一个上下文 embedding xix_ixi​ ，为了促进 BERT 模型训练和 fine-tuning 过程，我们重构了序列为 “[CLS]” + sentence + “[SEP]” + target mention + “[SEP]” 作为输入到 BERT 中。与 GloVe embeddings 相似，我们也采用 POS-tag embeddings 和 position embeddings 作为额外输入。对于基于 BERT 的模型，BiLSTM 模块就不必要了，因为 BERT embeddings 已经包含上下文特征，由于更好的模型设计和大规模的训练语料。

Relational Graph Attention Network

图形注意网络层在输入层表示上学习了一个深度表示，集成了 target 和上下文特征。我们的关系图注意力网络（RGAT）旨在根据句法依赖路径在单词之间进行信息交换。

我们首先介绍 baseline GAT model，它以 G=(V,A)G=(V,A)G=(V,A) 作为输入，然后再提出 RGAT 模型，它可以利用额外的关系特征用于更好的特征表示。
